## Plans and working files

All plan files generated by skills (grill-me, prd-to-issues, run-milestone, or any
other skill) must be written to `docs/plans/` inside this repository, never to
`~/.claude/plans/` or any other location outside the project root.

When a skill asks where to save a plan, always use:
  docs/plans/<descriptive-kebab-case-name>.md

If a plan file already exists at `docs/plans/<name>.md`, update it in place rather
than creating a new file.
